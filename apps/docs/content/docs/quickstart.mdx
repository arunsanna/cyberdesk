Get up and running with Cyberdesk in minutes.

## Fastest Start: Cyberdesk Starter Assistant UI

The quickest way to get started with Cyberdesk is to use our [Starter Assistant UI](https://github.com/cyberdesk-hq/cyberdesk-starter-assistant-ui) template. This ready-to-use Next.js application demonstrates how to build an AI assistant with virtual desktop control capabilities using the Cyberdesk API and Anthropic's Claude AI model.

### Features

- Interactive virtual desktop with streaming capabilities
- AI assistant chat interface powered by Claude 3.7 Sonnet
- Desktop control via AI (mouse clicks, keyboard input, screenshots)
- Bash command execution in the virtual environment
- Responsive design for both desktop and mobile devices

### Quick Setup

1. Clone the repository (alternatively, click "Use this template" on GitHub):

```bash
git clone https://github.com/cyberdesk-hq/cyberdesk-starter-assistant-ui.git
cd cyberdesk-starter-assistant-ui
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env.local` file with your API keys:

```
CYBERDESK_API_KEY=your_cyberdesk_api_key_here
ANTHROPIC_API_KEY=your_anthropic_api_key_here
```

4. Start the development server:

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

This template provides a complete working example that you can customize for your specific use case. For more details, check out the [repository README](https://github.com/cyberdesk-hq/cyberdesk-starter-assistant-ui).

## Building a Custom Integration with the TypeScript SDK

If you want to build your own integration, the **official TypeScript SDK** is the recommended approach. The rest of this guide shows how to use the SDK.

### Prerequisites

- Node.js installed
- A Cyberdesk API key (obtain from your [Cyberdesk dashboard](https://cyberdesk.io/dashboard))

### Installation

Install the SDK package:

```bash
npm install cyberdesk
# or
yarn add cyberdesk
# or
pnpm add cyberdesk
```

### 1. Initialize the Client

Create a Cyberdesk client instance with your API key:

```typescript
import { createCyberdeskClient } from 'cyberdesk';

const cyberdesk = createCyberdeskClient({
  apiKey: 'YOUR_API_KEY',
  // Optionally, you can override the baseUrl or provide a custom fetch implementation
});

console.log('Cyberdesk client initialized.');
```

### 2. Launch a Desktop Instance

Use the `launchDesktop` method to create a new virtual desktop:

```typescript
async function launch() {
  console.log('Launching desktop...');
  const launchResult = await cyberdesk.launchDesktop({
    body: { timeout_ms: 600000 } // Optional: 10-minute timeout
  });

  // Error handling - check for the error property
  if ('error' in launchResult) {
    console.error('Failed to launch desktop:', launchResult.error);
    throw new Error('Failed to launch desktop: ' + launchResult.error);
  }

  // Success - result contains id and status
  console.log('Desktop launch initiated:', launchResult);
  return launchResult.id; // Return the ID for the next steps
}

// We'll call this function later
```

*The response from `launchDesktop` contains the `id` and initial `status`.* 

### 3. Get Desktop Information (Including Stream URL)

Once the desktop is running (or pending), you can fetch its details, including the `stream_url`:

```typescript
async function getInfo(desktopId: string) {
  console.log(`Getting info for desktop ${desktopId}...`);
  // Poll until the desktop is running (optional, for demonstration)
  let info;
  for (let i = 0; i < 10; i++) { // Try up to 10 times
    info = await cyberdesk.getDesktop({ path: { id: desktopId } });
    if (!('error' in info) && info.status === 'running') {
      break; // Exit loop if running
    }
    if ('error' in info) {
      console.error('Error getting info:', info.error);
      throw new Error('Failed to get desktop info: ' + info.error);
    }
    console.log(`Status: ${info.status}, waiting...`);
    await new Promise(resolve => setTimeout(resolve, 5000)); // Wait 5 seconds
  }

  if (!info || 'error' in info) {
     throw new Error('Failed to get running desktop info.');
  }

  console.log('Desktop info retrieved:', info);
  console.log('Stream URL:', info.stream_url);
  return info;
}

// We'll call this function later
```

*The `stream_url` can be opened in a browser or used with a VNC client.* 

### 4. Control the Desktop

Interact with the running desktop using methods like `executeComputerAction` and `executeBashAction`.

#### Perform a Mouse Click

```typescript
async function clickMouse(desktopId: string) {
  console.log(`Clicking mouse on desktop ${desktopId}...`);
  const actionResult = await cyberdesk.executeComputerAction({
    path: { id: desktopId },
    body: {
      type: 'click_mouse',
      x: 100,
      y: 150,
      button: 'left' // Optional, defaults to left
    }
  });

  if ('error' in actionResult) {
    console.error('Mouse click failed:', actionResult.error);
    throw new Error('Mouse click failed: ' + actionResult.error);
  }

  console.log('Mouse click successful:', actionResult);
}

// We'll call this function later
```

#### Run a Bash Command

```typescript
async function runBash(desktopId: string) {
  console.log(`Running bash command on desktop ${desktopId}...`);
  const bashResult = await cyberdesk.executeBashAction({
    path: { id: desktopId },
    body: {
      command: 'ls -la /tmp'
    }
  });

  if ('error' in bashResult) {
    console.error('Bash command failed:', bashResult.error);
    throw new Error('Bash command failed: ' + bashResult.error);
  }

  console.log('Bash command output:', bashResult.output);
}

// We'll call this function later
```

### 5. Stop the Desktop Instance

When finished, use the `terminateDesktop` method to stop the instance and release resources:

```typescript
async function terminate(desktopId: string) {
  console.log(`Terminating desktop ${desktopId}...`);
  const terminateResult = await cyberdesk.terminateDesktop({
    path: { id: desktopId }
  });

  if ('error' in terminateResult) {
    console.error('Termination failed:', terminateResult.error);
    // Note: You might still want to proceed even if termination fails
  }

  console.log('Desktop termination result:', terminateResult);
}

// We'll call this function later
```

### Putting it all Together

```typescript
// (Make sure you have the client initialized as shown in Step 1)

async function main() {
  let desktopId: string | undefined;
  try {
    // 1. Launch
    desktopId = await launch();

    // 2. Get Info (wait for running state)
    await getInfo(desktopId);

    // 3. Control (Click and Bash)
    await clickMouse(desktopId);
    await runBash(desktopId);

    console.log('Quickstart complete!');

  } catch (error) {
    console.error('An error occurred during the quickstart:', error);
  } finally {
    // 4. Terminate (always attempt, even if errors occurred)
    if (desktopId) {
      await terminate(desktopId);
    }
  }
}

main(); // Run the quickstart sequence
```
